# Audio Assets

This directory contains placeholder audio files for testing.

## Weapon Sounds

- `uzi-fire.mp3` - Rapid SMG fire sound for Uzi
- `ak47-fire.mp3` - Rifle crack sound for AK47
- `shotgun-fire.mp3` - Deep boom sound for Shotgun

## Note

These are temporary placeholder files. Production-ready sound assets should be:
- Royalty-free or properly licensed
- Optimized for web delivery (compressed MP3 or OGG format)
- Normalized volume levels across all weapons
- Short duration (< 1 second) for rapid fire weapons
